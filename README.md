## ğŸ“˜ Usage

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Charley-xiao/manthano
cd manthano
```
2. Install dependencies:
```bash
cd frontend
npm install
cd ../backend
pip install -r requirements.txt
```

### Running the Application

1. Start the backend server:
```bash
cd backend
python server.py
```
2. Start the frontend:
```bash
cd frontend
npm run serve
```
